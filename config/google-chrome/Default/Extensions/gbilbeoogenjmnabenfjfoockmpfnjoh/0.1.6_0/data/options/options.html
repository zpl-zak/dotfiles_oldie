Night Mode Pro
Night Mode Pro is a browser extension to change the screen's luminosity (Day/Night modes) when you surf the Internet. Besides, you can easily adjust the screen's brightness.
Night Mode Pro protects your eyes by dimming the light when the environment is dark or vise-versa. In order to add a website to the white-list table, enter the website address in the input field below and click on (+) sign or press enter. Please note: the address should be a valid url (i.e. https://www.google.com/), otherwise it won't be added to the list.
